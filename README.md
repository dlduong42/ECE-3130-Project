## Morse Code Translator

### Project Purpose
This was created with the intent of allowing those who do not understand morse code to communicate with those who do by translating english text into morse code.

### Project Functions
This project takes input from a terminal and translates into morse code using an array, which can then be printed on an LCD screen through the STM32 Microcontroller.
